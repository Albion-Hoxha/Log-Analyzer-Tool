# Log-Analyzer-Tool
Simple log analyzer written in C++ for practice with file handling and string parsing.

## Features
- Reads log file
- Counts ERROR, INFO, WARNING messages
- Uses file handling
- Uses string parsing

## How to run

Compile:

g++ log_analyzer.cpp -o log_analyzer

Run:

./log_analyzer

## Example log file

INFO User login
ERROR Wrong password
WARNING Disk full
